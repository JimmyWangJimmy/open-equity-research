# Agent roles

Runtime task packets are generated under each company's `agent_tasks/` directory and include the current evidence ledger. These static files document the intended separation of duties.

- `fundamental.md`: bounded explanation of operating and financial evidence.
- `bear.md`: disconfirming analysis and missing evidence.
- `risk.md`: data, model, company, valuation, and portfolio risks.
- `verifier.md`: claim-to-evidence audit.

An LLM response is never a primary source. It may propose claims or questions, but verified source evidence must remain separate.
